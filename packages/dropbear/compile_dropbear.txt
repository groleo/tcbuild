dropbear_0.52 ssh server/client compilation

For arm compilation
export NM=/usr/local/arm/3.4.3-0.9.27/arm-linux/bin/nm \
LD=/usr/local/arm/3.4.3-0.9.27/arm-linux/bin/ld \
STRIP=/usr/local/arm/3.4.3-0.9.27/arm-linux/bin/strip \
LDD=/usr/local/arm/3.4.3-0.9.27/arm-linux/bin/ldd \
CC=/usr/local/arm/3.4.3-0.9.27/arm-linux/bin/gcc 

Change to dropbear directory
cd dropbear-0.52/

make clean

Run configure disabling pam, large file support, zip
./configure --disable-pam  --disable-largefile --disable-zlib

Compile programs:
make PROGRAMS="dropbear dbclient scp dropbearkey dropbearconvert" MULTI=0 STATIC=0 SCPPROGRESS=0

dropbear - server
dbclient - client
scp - secure copy
dropbearkey - keys generation 
dropbearconvert - convert openssh keys to dropbear

Discard all symbols, .comment and .not sections:
$STRIP -s -R .comment -R .note  dropbear dbclient dropbearkey dropbearconvert scp

Copy dropbear, dropbearkey, scp in /access_node/bin/.

Change current directory to /access_node/activity_files/
cd /access_node/activity_files/

Generate the rsa key and dsa key current directory (/access_node/activity_files/)
dropbearkey -t rsa -f dropbear_rsa_host_key
dropbearkey -t dss -f dropbear_dss_host_key
